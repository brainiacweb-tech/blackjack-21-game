# Blackjack / 21 Card Game

> Terminal Blackjack (21) — hit or stand to beat the dealer without going over 21.

---

## Features
- Hit or Stand mechanics
- Dealer auto-plays above 17
- Bust detection
- Replay support

## Requirements
```
No external dependencies — pure Python 3
```

## Installation
```bash
git clone https://github.com/brainiacweb-tech/blackjack-21-game
cd blackjack-21-game
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
